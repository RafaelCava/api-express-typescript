# Projeto API RestFull para Estudo

## Iniciar Projeto

 - para iniciar o projeto primeiro baixe as dependencias
```bash
npm install
#or
yarn
```
 - Após baixar as dependencias crie a build do projeto
```bash
npm run build
#or
yarn build
```
 - Depois bastar rodar esse script para iniciar a API
```bash
npm run start
#or
yarn start
```
